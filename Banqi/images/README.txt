角色頭像圖片放置說明
========================

請將角色圖片放在此資料夾中，檔名需與下方一致（副檔名可用 .png 或 .jpg）：

  劉備      → liubei.png   (或 liubei.jpg)
  關羽      → guanyu.png
  張飛      → zhangfei.png
  曹操      → caocao.png
  孫權      → sunquan.png
  黃月英    → huangyueying.png
  小喬      → xiaoqiao.png
  大喬      → daqiao.png

建議尺寸：至少 144×144 像素（正方形），遊戲中會顯示為圓形。

若未放置某角色的圖片，該角色會使用預設的線上頭像。
